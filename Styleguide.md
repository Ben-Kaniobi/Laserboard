Styleguide
==========

Constants & enums
-----------------
```
ALL_CAPS
```

Variables
----------------
* Global:```
Upper_case
```

* Local:```
lower_case
```

Functions
---------
```
Upper_case()
```

Curly brackets
--------------
* Only one action:
```
if(n == 1) Without_Curly_brackets();
```

* Multiple actions:
```
if(n > 1)
{
       Curly_brackets();
       Required();
}
```

* If with else:
```
if(Else_used)
{
       Also_use();
}
else
{
       Curly_brackets();
}
```
